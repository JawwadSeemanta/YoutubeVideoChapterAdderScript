1. Copy the timestamps from youtube description
2. Run the script
3. Copy the .pbf file to the video directory.
4. Rename it to the file name.
5. Open the file with PotPlayer.
6. Upon opening, go to 
		Right-Click_Menu > Playback > Chapter/Bookmark > Play bookmarks as seperate files in playlist
7. Done!!

Now you can navigate to the chapters with the help of playlist using the added bookmarks.
